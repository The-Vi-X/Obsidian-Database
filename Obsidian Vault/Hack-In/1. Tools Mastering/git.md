`git clone` - копирование репозитория
`git log [file]` - просмотр коммитов, которые затронули файл
`git show [commit]` - просмотр содержимого файла до обновления
`git add` - добавляет обновления в ветку
`git commit -m 'description'` - создание списка того, что ты изменил в определённое время
'git push' - отправка изменений в репозиторий
