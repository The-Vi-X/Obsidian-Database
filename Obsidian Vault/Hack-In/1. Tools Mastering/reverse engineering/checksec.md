#checksec
`checksec --file=binary` - проверяет файл на безопасность

RERLO - функция, делающая некоторые разделы файла доступными только для чтения
STACK CANARY - проверка на переполнение буфера
NX - функция запрета на одновременную запись, чтение и исполнение кода
PIE - программа для загрузки зависимостей программы в разные ячейки памяти для защиты проведения атак на определённые ячейки
