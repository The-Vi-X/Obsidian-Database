#UnserializeInjection


| Этап | Действия                                                                             |
| ---- | ------------------------------------------------------------------------------------ |
| 1    | Скопировать нужный класс локально в код                                              |
| 2    | Изменить переменные класса на публичные если нужно                                   |
| 3    | Создаём объект класса                                                                |
| 4    | Перезаписываем свойства переменных под наши нужды, проверяя, что они делают в классе |
| 5    | Серилизуем данные и выводим их                                                       |
| 6    | Вводим в форму для ввода unserialize данных наши серилизованные данные               |
