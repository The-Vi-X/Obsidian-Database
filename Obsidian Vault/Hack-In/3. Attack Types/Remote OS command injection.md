#ROSCI 
Атака на сайт, если есть возможность удалённо выполнить команду
Требуется преобразовать команду в URL-кодировку, чтобы она сработала

| Символ                                    | URL-кодировка | Зачем                             |
| ----------------------------------------- | ------------- | --------------------------------- |
| пробел                                    | `%20` или `+` | Разделение аргументов команды     |
| `&`                                       | `%26`         | Разделитель команд в shell        |
| `;`                                       | `%3B`         | Альтернативный разделитель        |
| `\|`                                      | `%7C`         | Pipe (передача вывода)            |
| `#`                                       | `%23`         | Комментарий в shell / якорь в URL |
| `$`                                       | `%24`         | Подстановка переменных            |
| `` ` `` \| `%60` \| Выполнение команды \| |               |                                   |
| `/`                                       | `%2F`         | Разделитель путей                 |
| `\`                                       | `%5C`         | Экранирование                     |
| `?`                                       | `%3F`         | Wildcard / начало query string    |
| `*`                                       | `%2A`         | Wildcard                          |
| `(` `)`                                   | `%28` `%29`   | Группировка команд                |
| `<` `>`                                   | `%3C` `%3E`   | Перенаправление ввода/вывода      |
| `!`                                       | `%21`         | История команд / отрицание        |
| `"`                                       | `%22`         | Кавычки                           |
| `'`                                       | `%27`         | Кавычки                           |
| `\n`                                      | %0A           | Перенос строки                    |
